### Golden Garlic

홈랩으로 프로덕션급 인프라를 직접 설계·운영하며 배우는 인프라 엔지니어입니다.
Kubernetes와 가상화, 그리고 그것을 관측 가능하게 만드는 일에 관심이 많습니다.

**무엇을 하나**
- Self-hosted homelab: Kubernetes(Talos / Omni), VMware vSphere
- 통합 observability 스택 직접 구축·운영 (메트릭 / 로그 / 알림)
- 홈랩 공용 SSO(Keycloak), 선언적 인프라(IaC)

**기술 스택**
- Orchestration: Kubernetes, Talos Linux
- Virtualization: VMware vSphere / ESXi
- Observability: Grafana, VictoriaMetrics(Prometheus), Loki, Alertmanager
- Infra: Docker, Caddy, Keycloak, Go
